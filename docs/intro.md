## About LearningHubz

LearningHubz is a learning platform built with Next.js 14, featuring:
- **Next.js App Router** - Modern React framework with server components
- **Azure Cosmos DB** - Global distributed database
- **Azure Search** - Powerful search capabilities
- **SignalR** - Real-time communication
- **Multi-language support** - Internationalization with next-intl
